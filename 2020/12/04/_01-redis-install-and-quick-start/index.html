# 1. Redis 下载

# For Windows:

https://github.com/MicrosoftArchive/redis/releases/download/win-3.2.100/Redis-x64-3.2.100.msi

# 2. Redis 安装

双击 Redis-x64-3.2.100.msi. 然后点击 Next.

阅读许可条款并同意。然后点击 Next.

设置安装目录，将路径添加到环境变量中。然后点击 Next.

设置端口（保持默认）. 然后点击 Next.

设置内存限制（保持默认）.然后点击 Next.

点击 Install.

点击 Final.

打开任务管理器，点击 Services 选项卡，查看已经运行的 Redis 服务。

介绍常见的可执行文件。

  • redis-cli 是与 Redis 对话的命令行接口工具。
  • redis-server.exe 是 Redis 服务器本身。
  • redis-benchmark 是用来检查 Redis 性能的。

# 3. Redis 入门

检查 Redis 是否工作。

运行 redis-cli，后面跟一个命令名和它的参数，就会把这个命令发送到在 localhost 上运行的 Redis 实例，端口为 6379。你可以改变 redis-cli 使用的主机和端口，尝试使用 --help 选项来查看使用信息。

另一种有趣的方式是在没有参数的情况下运行 redis-cli：程序会在交互模式下启动，你可以输入不同的命令并查看它们的回复。

tips

常见的命令行用法：https://redis.io/topics/rediscli

redis-cli 是 Redis 命令行接口，是一个简单的程序，可以直接从终端向 Redis 发送命令，并读取服务器发送的回复。

Redis 配置：https://redis.io/topics/config
查看如何配置 Redis

恭喜你，你可以开始使用 Redis 了。
