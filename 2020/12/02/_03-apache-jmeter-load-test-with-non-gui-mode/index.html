# 2. 使用命令运行 Test Plan

在命令提示符下使用命令去运行 Test Plan ，如下：

jmeter -n -t [jmx file] -l [results file]

# -n: non-GUI mode
# -t: location for jmeter test script
# -l: Location of the result file

# 2. 生成报告

JMeter 支持从 Test Plan 生成 dashboard 报告并获取图像和统计。

dashboard 生成器是 JMeter 的一个模块化扩展。在负载测试结束或者需求结束的时候，它默认读取和处理 CSV 文件中的数据，去生成包含图像视图的 HTML 文件。

在命令提示符下使用命令去运行 Test Plan 并生成报告，如下：

jmeter -n -t [jmx file] -l [results file] -e -o [Path to web report folder]

D:\Program Files\apache-jmeter-5.3\bin
# -e: To generate HTML Reports
# -o: Location of the Output folder
