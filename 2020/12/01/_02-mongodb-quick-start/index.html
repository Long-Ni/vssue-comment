# 1. MongoDB 入门

# 1.1. 启动与连接

  1. 添加 MongoDB 服务端为 Windows 服务, 需要指定配置文件（数据库存储目录，log 存储目录）

  2. 启动 MongoDB 服务

  3. 设置 MongoDB 全局变量

  4. 连接 Mongo

    mongo 127.0.0.1:27017

# 1.2. MongoDB 常用命令

  • db: 显示当前所在数据库，默认为 test
  • show dbs: 列出可用数据库
  • show collections: 列出数据库中可用集合
  • use: 用于切换数据库

Mongo DB 预设有两个数据库， admin 和 local。

admin 用来存放系统数据， local 用来存放该实例数据。

可以 use 一个不存在的数据库，当你存入新数据时， MongoDB 会创建这个数据库。

> use newDB
switched to db newDB
> db.newCollection.insert({x:1})
WriteResult({ "nInserted" : 1 })
>

以上命令向数据库插入一个文档，返回 1 表示插入成功，MongoDB 自动创建 newCollection 集合和数据库 newDB。

# 1.3. MongoDB CRUD 操作

# 1.3.1. Create 操作

MongoDB 提供 insert() 方法创建新文档。

  • db.collection.insertOne() 插入单个文档
  • db.collection.insertMany() 插入多个文档
  • db.collection.insert() 插入单条或者多条文档
> db.newCollection.insertMany([{name: "awesome_1", age: 18}, {name: "awesome_2", age: 19}])
{
        "acknowledged" : true,
        "insertedIds" : [
                ObjectId("5faba241a461e0c96bb5a56c"),
                ObjectId("5faba241a461e0c96bb5a56d")
        ]
}
>

# 1.3.2. Read 操作

MongoDB 提供 find() 方法查找文档。第一个参数（query criteria）为查询条件，第二个参数（projection）可以传入查询结果想要显示的字段名称。

> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
> db.newCollection.find({name: "awesome"});
> db.newCollection.find({name: "awesome_1"});
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
> db.newCollection.find({age: {$gt: 18}});
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
> db.newCollection.find({age: {$gt: 18}},{name:1});
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2" }
> db.newCollection.find({age: {$gt: 18}},{name:1, _id:false});
{ "name" : "awesome_2" }

指针（Cursor）方法： count()、 skip()、 limit()、sort()、 pretty() 等，改变文档查询的执行方式。

> db.newCollection.find().count();
3
> db.newCollection.find().skip(1).limit(1).sort({age: 1})
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
> db.newCollection.find().skip(1).limit(2).sort({age: 1})
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
> db.newCollection.find().skip(1).limit(2).sort({age: -1})
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
>

# 1.3.3. Update 操作

MongoDB 提供 update() 方法更新文档。

  • db.collection.updateOne() 更新最多一个符合条件的文档
  • db.collection.updateMany() 更新所有符合条件的文档
  • db.collection.replaceOne() 替换最多一个符合条件的文档
  • db.collection.update() 默认更新一个文档。可通过配置 multi 参数，更新多个文档。
> db.collection.update(
	<query>, // query 为查询条件
	<update>, // update 为修改的文档
	{
		upsert: <boolean>, // 为真，查询为空时插入文档。The default value is false.
		multi: <boolean> // 为真，更新所有符合条件的文档。The default value is false.
	}
)

需要注意的是，如果更新文档只传入 age 字段，那么文档会被更新为 {age: 15}，而不是 {name: "awesome_1", age: 15}。

使用 $set 指令，避免文档被覆盖， $set 仅替换或添加指定字段。

> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
> db.newCollection.update({name:"awesome_1"},{name:"awesome_1",age:15})
WriteResult({ "nMatched" : 1, "nUpserted" : 0, "nModified" : 1 })
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 15 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
> db.newCollection.update({name:"awesome_1"},{$set:{age:18}})
WriteResult({ "nMatched" : 1, "nUpserted" : 0, "nModified" : 1 })
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
> db.newCollection.update({name:"awesome_3"},{$set:{age: 20}},{upsert:true})
WriteResult({ "nMatched" : 0, "nUpserted" : 1, "nModified" : 0, "_id" : ObjectId("5fac9d835fd971dc846e8de7") })
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19 }
{ "_id" : ObjectId("5fac9d835fd971dc846e8de7"), "name" : "awesome_3", "age" : 20 }
> db.newCollection.update({age: {$gt: 15}},{$set:{test: "A"}},{multi: 1})
WriteResult({ "nMatched" : 3, "nUpserted" : 0, "nModified" : 3 })
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18, "test" : "A" }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19, "test" : "A" }
{ "_id" : ObjectId("5fac9d835fd971dc846e8de7"), "name" : "awesome_3", "age" : 20, "test" : "A" }
>

# 1.3.4. Delete 操作

MongoDB 提供 delete() 方法删除文档。

  • db.collection.deleteOne() 删除最多一条符合条件的文档
  • db.collection.deleteMany() 删除所有符合条件的文档
  • db.collection.remove() 删除一个或者多个文档
  • db.collection.drop() 删除集合，返回 true 表明删除成功
> db.collection.remove(
   	<query>, // To delete all documents in a collection, pass an empty document ({}).
   	{
     	justOne: <boolean>, // Omit to use the default value of false and delete all documents matching the deletion criteria.
     	writeConcern: <document>,
     	collation: <document>
   	}
)

示例：

> db.newCollection.remove({name: "awesome_3"})
WriteResult({ "nRemoved" : 1 })
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18, "test" : "A" }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56d"), "name" : "awesome_2", "age" : 19, "test" : "A" }
> db.newCollection.remove({age: {$gt: 19}})
WriteResult({ "nRemoved" : 0 })
> db.newCollection.remove({age: {$gt: 18}})
WriteResult({ "nRemoved" : 1 })
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18, "test" : "A" }
> db.newCollection.find();
{ "_id" : ObjectId("5fab9e50a461e0c96bb5a56b"), "x" : 1 }
{ "_id" : ObjectId("5faba241a461e0c96bb5a56c"), "name" : "awesome_1", "age" : 18, "test" : "A" }
>

# 1.4. 小结

相比传统的关系型数据库，MongoDB 的 CRUD 操作更像是编写程序，更符合开发人员的直觉，不过 MongoDB 同样也支持 SQL 语言。

MongoDB 的 CRUD 引擎配合索引技术、数据聚合技术和 JavaScript 引擎，赋予 MongoDB 用户更加强大的操纵数据能力。
